# Flappy-Blink
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

![](./public/shreenshot.png)

## Develop
```bash
# install dependencies
$ npm i
# serve with hot reload at localhost:3000
$ npm run start
```

## Build
``` bash
# install dependencies
$ npm i

# build for production
$ npm run build
```
